# Keyboard Shortcuts

## Hierarchy navigation and reordering

These shortcuts work when a file chip in the hierarchy is selected (click a chip to select it).

| Key | Action |
|-----|--------|
| ↑ | Move the selected file up |
| ↓ | Move the selected file down |
| → | Nest under the file above (make it a child) |
| ← | Unnest (move up one level) |

Movement crosses level boundaries — pressing ↑ at the top of a group moves the file above its parent.

## Unlinked file shortcuts

These shortcuts work when one or more files in the Unlinked pane are selected.

| Key | Action |
|-----|--------|
| ↑ | Move the selected file up in the Unlinked list |
| ↓ | Move the selected file down in the Unlinked list |
| ← | Move all selected files into the hierarchy |

Using ↑ or ↓ in the Unlinked pane automatically switches the sort order to Custom.

## Editor shortcuts

| Key | Action |
|-----|--------|
| Ctrl+S / Cmd+S | Save the current file |
| Escape | Close the editor |

## Renaming

| Action | How |
|--------|-----|
| Rename a file chip | Double-click the chip label |
| Rename from the editor | Double-click the filename in the editor toolbar |
| Rename a project | Double-click the project chip label |
