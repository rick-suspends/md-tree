# Introduction

Welcome to the **Markdown Collection Editor**.

This is a sample document. You can edit this file directly in the editor, preview the HTML output, and organize documents using the sidebar.

## Features

- Sidebar with filename or H1 title view
- Drag and drop to reorder and nest files
- Edit markdown with **vi keybindings**
- Live HTML preview
- Edit `collection.yaml` directly

## Getting Started

Use the sidebar to navigate between files. Drag files to reorder or nest them under other documents.

```bash
# Start the app
./start.sh
```
